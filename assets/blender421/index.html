Instalación de Blender en portable
Descargar Blender 4.2.1 en el botón MacOS, Linux and.. Aquí
Buscamos la opción de portable zip y descargamos
Creamos en el equipo de computo una carpeta llamada blender421 y ahí...
...Descomprimimos el archivo zipeado
ejecutamos el archivo blender.exe
Ya estando en el programa le damos esc, para que desaparezca el Splash screen
Buscamos el menu Edit >> Preferences y vamos a configurar algunos parámetros del programa:
Interface >> Language >> Spanish >> Tooltips
Interface >> Addons >> Node Wrangler
Interface >> Navigation >> Orbit Around Selecion, Perpective, Depth
Interface >> Navigation >> Zoom to Mouse Position
Interface >> Input >> Emulate Numpad *solo cuando no tengamos teclado numérico
Interface >> System >> Activar la tarjeta gráfica si se tiene.
interface >> System >> Memory & Limits >> Undo Steps >> 256
Interface >> Auto-Save validar que este activado
Finalizamos
